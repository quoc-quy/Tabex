2 Installation

Method Steps
Direct Download Download ZIP

Simply extract the ZIP file and use the tabex.min.js files in your project.

Tabex provides a small JS file that you can include directly via a CDN link. This makes it easy to integrate into your project and customize as needed.

Include the Tabex JavaScript file in your project:

                
<script src="https://cdn.jsdelivr.net/gh/quoc-quy/Tabex@v1.0.0/tabex.min.js"></script>

You can use the released versions of Tabex available here.

3 Usage

Here’s the basic HTML structure you’ll need to set up Tabex:

                    
<ul id="my-tabs">
    <li><a href="#panel1">Tab 1</a></li>
    <li><a href="#panel2">Tab 2</a></li>
</ul>
<div id="panel1">Content 1</div>
<div id="panel2">Content 2</div>

And here’s how you can use it with JavaScript:

                    
// Initialize a new tab instance
const tabs = new Tabex('#my-tabs', {
    activeClassName: 'tabex--active',
    remember: true, // Keeps the active tab in the URL
    onChange: function({ tab, panel }) {
        console.log(`Switched to ${tab.textContent}`);
    }
});

// Switch to a specific tab
tabs.switch('#panel2');

// Destroy the tab instance
tabs.destroy();

4 Options

Option Type Description
activeClassName string OCSS class applied to the active tab (default: 'tabex--active' )
remember boolean Persists the active tab state in the URL (default:false)
onChange function A callback function that runs whenever the active tab changes. It receives an object with two properties: tab (the DOM element of the newly active tab) and panel (the DOM element of the corresponding content panel). By default, it’s set to null , meaning no action is taken unless you define a function.

5 API

Method Description
switch(input) Switches to a specific tab based on the provided input . The. input can be either: (1) a string representing the '#panel2' attribute of a tab (e.g., '#panel2' ), or (2) a DOM element that is one of the tab links (e.g., an <a> element from the tabs list).
destroy() Removes all Tabex functionality, restores the original HTML, and unbinds events.
